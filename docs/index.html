chunk-buffer

Simple function to chunk a buffer into an array of buffers.

chunk-buffer(buffer: Buffer, size: number): Array<Buffer>
Parameters
buffer (Buffer) The buffer to be chunked.
size (number) The chunk size.
Returns
Array<Buffer>: The chunked Buffer.
Example
chunkBuffer(Buffer.from([
	1,
	2,
	3,
	4,
	5
]), 3);

// returns [Buffer<1,2,3>, Buffer<4,5>]
